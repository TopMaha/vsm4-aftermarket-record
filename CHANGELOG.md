### 2) WIP IN 1.5 — ค้นหาด้วย Valve No / Lot

- แผง "ลืมลงบันทึก WIP IN 1.5" (หน้าตั้งค่า): เพิ่มช่องค้นหา Valve No / Lot (กรองสดขณะพิมพ์)

### 3) WIP IN 1.5 — ปุ่มแก้ไข (กันพนักงานลง Valve No ผิด)
- **ตาราง WIP IN (ป๊อปอัป "รอผลิต")**: เพิ่มปุ่ม **✏️** ต่อรายการที่ยังรอผลิต → แก้ Valve No / Lot / จำนวน แล้วอัปเดตขึ้น Cloud (LWW) — รายการที่เคยลงผิดแล้วค้าง จะ match ยอดผลิตและหลุดออกจาก "รอผลิต"
- **แผง "ลืมลงบันทึก WIP IN 1.5"**: เพิ่มปุ่ม **✏️ แก้** → แก้ Valve No / Lot ที่พนักงานลงผิด "ตอนบันทึกผลิต" ที่ต้นทาง (อัปเดตทุก KPI + Production record ของ Valve/Lot นั้น ทั้ง Local + Cloud) แล้วรายการหายจากลิสต์

### 4) การบันทึก — เพิ่มช่วงเวลาแล้ว "บันทึกไม่ได้/ไม่หมด" ให้ชัดขึ้น

- เพิ่มปุ่ม **🗑 ลบช่วงนี้** ในหน้าต่างแจ้งเตือน — สำหรับ "ช่วงที่กดเพิ่มเข้ามา" (ช่วงหลักลบไม่ได้) กดลบช่วงที่ไม่ใช้/กรอกไม่ครบได้เลยแล้วกดบันทึกอีกครั้ง
- ปุ่ม "✏️ กลับไปแก้ไข" เลื่อนจอไปที่เครื่อง/ช่วงแรกที่ยังไม่ครบให้อัตโนมัติ

- ยอดจะไปตกวัน "เมื่อวาน / รวมทั้งเดือน" ในการ์ด IOT และคอลัมน์วันที่ในตาราง WIP IN ตามวันผลิตจริง — ถ้า Lot ผลิตหลายวัน ใช้วันผลิตล่าสุด
- หน้าต่างยืนยันบอกชัด "📅 ลงตามวันที่ผลิตที่ SSG (ไม่ใช่วันที่กด)" + รายการ preview มีวันที่กำกับ · ป้ายในแผงเปลี่ยนจาก "ล่าสุด" เป็น "วันที่ผลิต"
- ฟอร์มกรอกเอง "เข้ายอด WIP IN 1.5" ยังใช้วันที่ปัจจุบันตามเดิม · ระบบ sync ข้ามเครื่องไม่กระทบ (`updatedAt` ยังเป็นเวลาจริง)

### 2) Lot ยังไม่ลงทะเบียน — ปุ่ม "➕ ลงทะเบียนทั้งหมด" (ออก ID ให้อัตโนมัติ)

- มีหน้าตรวจทานก่อนยืนยัน (ติ๊กเลือก/ออกได้ · ปุ่มเลือกทั้งหมด/ล้างเลือก) — รายการที่ระบบเดาว่า**พิมพ์ผิด** (เช่น `$`→`S`) จะ**ไม่ถูกติ๊กไว้** พร้อมป้าย ⚠️ ให้กดแก้ Lot ก่อน
- ID Code ออกให้อัตโนมัติ `A000001…` (ตัวอักษรนำหน้า ไม่ชนบาร์โค้ดจริงที่เป็นเลขล้วน) + หมายเหตุ "ลงทะเบียนอัตโนมัติ…" ทุกแถว — ค้น/แก้/ลบทีหลังได้ในตาราง Valve IDs
- ขึ้น Cloud อัตโนมัติทีละ 8 รายการพร้อมแถบสถานะ · ลงทะเบียนแล้ว Lot หายจากแผงค้างและโผล่ในตัวเลือก Lot ของฟอร์มบันทึกทันที
